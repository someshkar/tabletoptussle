The Vision

The idea is to use the power of Augmented Reality to make a game where players can compete with each other in real time over any convenient surface, and all they need is any static object to track.

Game Dynamics

The game consists of up to four players per session competing with tanks, which are customizable and allow for modifications via in-game currency. The game functions similarly to MortalKombatTM but is AR based and has mechanical vehicles instead of humans due to the inherent complications incurred by their use in AR environments and the aesthetic pleasure that one gets out of seeing cool vehicles.
